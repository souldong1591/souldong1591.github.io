一些常用的环境配置总结(Java、MySQL、Tomcat等等)

  • 环境配置

总结一些常见的环境配置，例如java、mysql等等，以备不时之需。

另外，配环境、安装开发工具有两个比较好的习惯：
1.路径不能有中文。
2.路径不能有空格(平时会默认装到 Program Files 文件夹，所以最好自己新建一个英文无空格的文件夹)

Java

  1. 到官网下载java的jdk安装包，下载jdk即可，它包含一个通用的jre。
  2. 一路next安装。完成后打开环境变量窗口，新建变量名：JAVA_HOME，变量值：D:\workspace\Java\jdk1.7.0(根据自己的安装目录来写)
  3. 打开PATH，添加变量值：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin
  4. 新建变量名：CLASSPATH，变量值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar（注意开始有个点）
  5. 在CMD命令下输入javac，java，javadoc命令：出现图示界面，表示安装成功。

备注：
1，.表示当前路径，%JAVA_HOME%就是引用前面指定的JAVA_HOME；
2，JAVA_HOME指明JDK安装路径，此路径下包括lib，bin，jre等文件夹，tomcat，eclipse等的运行都需要依靠此变量。
3，PATH使得系统可以在任何路径下识别java命令。
4，CLASSPATH为java加载类(class or lib)路径，只有类在classpath中，java命令才能识别。

MySQL

mysql官网提供了两种不同的安装方式，一种是 .msi 安装包，一种是 zip 压缩包。.msi 安装包比较简单，直接安装即可，注意下安装时的提示，按提示设置就行了。下面主要写一下 zip 压缩包的安装方法。

  1. 将下载的mysql压缩包解压
  2. 添加环境变量，选择PATH,在其后面添加: 你的mysql bin文件夹的路径 (如:D:\workspace\MySQL\MySQL Server 5.6\bin，注意是追加,不是覆盖)
  3. 在mysql根目录找到my-default.ini文件，复制一份，将其改为my.ini，打开my.ini，在其中修改或添加配置：

4.以管理员身份运行cmd（一定要用管理员身份运行，不然权限不够），cd到mysql的bin文件夹下(不管有没有配置过环境变量，也要进入bin文件夹，否则之后启动服务仍然会报错误2)

  1. 输入mysqld -install(如果不用管理员身份运行，将会因为权限不够而出现错误：Install/Remove of the Service Denied!)
  2. 安装成功后就要启动服务了，继续在cmd中输入:net start mysql，服务启动成功！

    注意：这个时候经常会出现错误2和错误1067。

  • 如果出现错误2 系统找不到文件，检查一下是否修改过配置文件或者是否进入在bin目录下操作，如果配置文件修改正确并且进入了bin文件夹，需要先删除mysql（输入 mysqld -remove）再重新安装（输入 mysqld -install）；
  • 如果出现错误1067，那就是配置文件修改错误，确认一下配置文件是否正确。

7.登录MySQL。登录MySQL的命令是mysql， mysql 的使用语法如下：
　　 mysql [-u username] [-h host] [-p[password]] [dbname]
username 与 password 分别是 MySQL 的用户名与密码，mysql的初始管理帐号是root，没有密码，注意：这个root用户不是Linux的系统用户。MySQL默认用户是root，由于初始没有密码，第一次进时只需键入mysql即可

  1. MySQL默认没有密码，安装完毕增加密码的重要性是不言而喻的。
    　 命令：mysqladmin -u root password 'new-password'
    　 格式：mysqladmin -u用户名 -p旧密码 password 新密码
    　 例：给root加个密码123456。键入以下命令 ：
    　　 mysqladmin -u root password 123456

    注：因为开始时root没有密码，所以-p旧密码一项就可以省略了

Tomcat

Tomcat需要先配置好Java环境。

  1. 必不可少的去官网下载，然后解压到一个文件夹内。
  2. 配置环境变量：新建变量名：CATALINA_BASE，变量值：C:\tomcat(改为自己的路径)
  3. 配置环境变量：新建变量名：CATALINA_HOME，变量值：C:\tomcat(改为自己的路径)
  4. 打开PATH，添加变量值：%CATALINA_HOME%\lib;%CATALINA_HOME%\bin
  5. 启动方法两种：
    1，方法一：在CMD命令下输入命令：startup，出现如下对话框，表明服务启动成功。
    2，方法二：右键点击桌面上的我的电脑->管理->服务和应用程序->服务，找到Apache Tomcat服务，右键点击该服务，选择属性，将启动类型由手动改成自动。

6.测试：打开浏览器，在地址栏中输入http://localhost:8080回车，如果看到Tomcat自带的一个JSP页面，说明你的Tomcat已搭建成功。
